## What is Ubik?

Ubik is a full-stack **motel booking and management platform** built as a Progressive Web App (PWA). It connects motel owners who want to list and manage their properties with clients looking to browse availability and make secure reservations — all from a single, mobile-first interface.

The platform is backed by a RESTful API (`https://ubik-back.duckdns.org/api`) and supports three distinct user roles with granular permissions, interactive maps, and integrated Stripe payments.

## Key features

<CardGroup cols={2}>
  <Card title="Interactive map exploration" icon="map-location-dot">
    Leaflet-powered maps show motel locations relative to the user's geolocation in real time.
  </Card>
  <Card title="Role-based access control" icon="lock">
    Three roles — Admin, Owner, and Client — each with tailored views and route-level permission guards.
  </Card>
  <Card title="Secure payments" icon="credit-card">
    Stripe integration handles reservation payments with success and failure redirect flows.
  </Card>
  <Card title="Owner dashboard" icon="gauge-high">
    Owners can create motels, add rooms with amenities, manage offers, and review reservations.
  </Card>
  <Card title="Progressive Web App" icon="mobile-screen">
    Installable on any device. The mobile-first UI adapts seamlessly from phone to desktop.
  </Card>
  <Card title="Server-side rendering" icon="server">
    Angular Universal + Express delivers fast initial page loads and improved SEO.
  </Card>
</CardGroup>
