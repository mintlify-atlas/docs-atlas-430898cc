## Welcome to Ubik

Ubik is a full-stack progressive web application (PWA) built with Angular that modernizes the motel booking experience in Colombia. It connects two types of users — **motel owners** who manage their properties online, and **clients** who discover and book rooms with ease.

## What is Ubik?

Ubik digitizes the motel sector by providing a modern, secure, and mobile-first reservation platform. It was designed to serve three distinct roles with different access levels and capabilities:

<CardGroup cols={3}>
  <Card title="Clients" icon="user">
    Search motels, view rooms, make reservations, and pay securely online.
  </Card>
  <Card title="Owners" icon="key">
    Register establishments, manage rooms and offers, and track bookings from a dedicated dashboard.
  </Card>
  <Card title="Admins" icon="shield-check">
    Oversee platform activity, approve or reject motel listings, and manage all users.
  </Card>
</CardGroup>

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend Framework | Angular 20 with TypeScript |
| Styling | Tailwind CSS 4 + PrimeNG |
| Maps | Leaflet.js |
| Payments | Stripe.js |
| Image Storage | Cloudinary |
| SSR | Angular Universal (Express) |
| State Management | RxJS + Angular Signals |
